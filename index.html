Grunt Boilerplate

This is a simplified version of Chris Coyiers' Grunt Boilerplate based on the CSS Tricks screencast First Moments with Grunt which also accompanies the article written by Chris, Grunt for People Who Think Things Like Grunt are Weird and Hard.

Usage

Clone the repository (using either HTTPS, SSH or Subversion):

git clone https://github.com/adelineyaw/GruntBoilerplate.git

Run the NPM install:

npm install

Run Grunt:

grunt
