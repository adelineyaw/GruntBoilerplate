Grunt Boilerplate

This is a simplified version of Chris Coyiers' Grunt Boilerplate based on the CSS Tricks screencast First Moments with Grunt which also accompanies the article written by Chris, Grunt for People Who Think Things Like Grunt are Weird and Hard.

Usage

Clone the repo to your project folder:

git clone https://github.com/adelineyaw/GruntBoilerplate.git

Then type:

npm install

And the npm will install all the Grunt plugins into the node modules folder.

For more information on each Grunt plugin listed in this boilerplate, visit the following links:

  • grunt-contrib-concat
  • grunt-contrib-imagemin
  • grunt-contrib-jshint
  • grunt-contrib-sass
  • grunt-contrib-uglify
  • grunt-contrib-watch

Once all the plugins have been installed, simply type:

grunt
